3. Methodology / rubric / reproducibility (from the PDF, formalised)

  • Dataset: SAVIOR-Bench, 509 expert-annotated financial docs — UCSF

Industry Documents 46.95% · in-house synthetic 17.88% · misc public financial 17.49% · Inv3D 9.04% · FUNSD 8.64%. IAA 0.761 (fuzzy string similarity, 187 double-annotated pairs).

  • Rubric: word-level recall & precision (multiset; chosen over

char-level because downstream LLM extraction needs whole tokens — "Totl" ≠ "Total"); PaIRS (pairwise spatial relationships, quadratic spatial penalty, z-scored) for layout fidelity, PC-models only.

  • Reproducibility: LoRA (rank 32, α 64) on vision-encoder final layer +

language decoder, Qwen2.5-VL-3B/7B-Instruct; lr 1e-4 (vision) / 5e-4 (language); batch 1, grad-accum 10; 7 (3B) / 5 (7B) epochs; 2× H100 80 GB, CUDA 12.3; inference top-p 0.9, temp 0.1; frozen 10-step layout prompt; baselines native, no fine-tune. Benchmark access on request: niyati.chhaya@hyprbots.com.

  • Factory invariant added on top: every published cell carries a

one-line rerun_command + passes repro_check (metric + CI + git commit match) under the deterministic harness — the provided numbers are re-derived here before they leave the lab.
