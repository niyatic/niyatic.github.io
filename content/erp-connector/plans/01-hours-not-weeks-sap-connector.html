3.2 Depth — connector expertise behind the speed

The depth pitch is why this is hours, not weeks:

  • OData v4 vs BAPI shapes — the descriptor abstracts the wire-format

choice; the agent picks target="sap_s4hana_odata" vs ..._bapi from the registry without touching the wire code.

  • Master-key resolution as policy, not code — master_key_resolver is a

lambda on the descriptor (docs/USAGE.md); the agent supplies the lookup, not the protocol.

  • Idempotency keyed to SAP's own uniqueness — `BusinessKey(["BUKRS",

"BELNR", "GJAHR"]) lines the framework's first line of dedup up with SAP's second line; both resolve to skipped_dup`, never an error.

  • Replay-safety formally proven — re-run on same source bytes posts 0

and emits a byte-identical artifact (check_replay_safety).

  • No HITL — low-confidence records dead-letter with raw response; never

a human queue (docs/LIMITS.md).
