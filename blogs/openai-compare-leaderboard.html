Method (pinned — identical to claude-compare)

  • Dataset: dataset_95_granular — 95 documents (82 PDF + 10 PNG + 3 JPG), each with reviewed gold JSON.
  • Prompt: the single fixed extraction query + output schema from /tmp/claude-compare/EXTRACTION-SPEC.md, identical across every model.
  • Decoding (pinned): temperature=0, max_output_tokens=8000.
  • Input: PDF → PNG via PyMuPDF @200 DPI → image input; PNG/JPG sent directly. One JSON output per doc in the gold {"parse":{"summary":...,"line_items":...}} schema.
  • Scorer: /tmp/dataset_95/harness/score.py — per-field P/R/F1, overall micro-F1, line-item-count exact accuracy, 95% bootstrap CI (2000 resamples, seed 20260522), n.
