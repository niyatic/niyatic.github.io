Classify the escrow statement period -- typed, in one call

hyperapi-classify-escrow-statement-period-tr turns a manual judgement on a escrow statement into one function call that returns a typed result plus a confidence gate.

Fails safe, never a queue

When the model is weak it re-runs automatically on a stronger profile. Documents that stay weak come back trustworthy=false with the full payload, for your automation to route by rule. It is never queued to a person.

Five lines

``python from hyperapi_classify_escrow_statement_period_tr import classify_file r = classify_file("doc.pdf") print(r.trustworthy) ``

Accuracy is published at apis.hyperbots.com. The model is an open-source document model fine-tuned by Hyperbots (Hyperbots IP); this repo is the typed client around it.
