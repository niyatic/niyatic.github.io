DC-000 · Parse — a three-facet task

The SAVIOR-Bench parse task is defined as three facets: (1) extraction — field/value/line-item + OCR text-recognition accuracy; (2) layout awareness — reading order, layout-element structure, table/region structure; (3) bounding box — visual grounding, IoU of predicted vs gold field/region boxes. Each facet declares its own metric(s). Only the extraction facet has data today; the other two are defined but not yet measured — no numbers are fabricated.
