1. Dataset — SAVIOR-Bench v1

  • Size: n = 509 expert-annotated documents.
  • Source mix (composition by share of the 509):

| Source | Share | |---|---| | UCSF Industry Documents | 46.95% | | In-house synthetic | 17.88% | | Misc. public financial | 17.49% | | Inv3D | 9.04% | | FUNSD | 8.64% |

  • Document classes: invoices, packing slips, forms, multilingual

scans, low-quality PDFs, complex tables.

2. Annotation quality — inter-annotator agreement (IAA)

  • IAA: 0.761 — computed as fuzzy string similarity over **187

double-annotated pairs**.

3. Latency (SAVIOR-Train fine-tuned inference path)

  • p50 latency: 2,548 ms
  • p99 latency: 3,319 ms
  • Failure rate: 3% (KV-cache-bound; a 2-GPU path is the noted

mitigation).

The PDF provides only these distribution points (p50, p99, failure rate). No full histogram and no p90/p999 are given in the source, so none is asserted downstream.

4. Cost (SAVIOR-Train inference path)

Infrastructure-derived on an A100 40 GB at ~$2/hr (not a price quote):

  • Cost per attempted call: $0.000186
  • Cost per successful call: $0.000191
  • ≈ $0.19 per 1,000 successful calls

The attempted-vs-successful gap reflects the 3% failure rate.

5. Fine-tuning (LoRA) recipe

  • Method: LoRA, rank 32, α 64, applied to the **vision-encoder

final layer + the language decoder**.

  • Backbones: Qwen2.5-VL-3B-Instruct and Qwen2.5-VL-7B-Instruct

(open-source; fine-tuned by Hyperbots → "Hyperbots-IP").

  • Learning rates: 1e-4 (vision) / 5e-4 (language).
  • Batching: batch size 1, gradient accumulation 10.
  • Epochs: 7 (3B) / 5 (7B).
  • Hardware: 2× NVIDIA H100 80 GB, CUDA 12.3.
  • Inference: top-p 0.9, temperature 0.1; frozen 10-step layout prompt.
  • Baselines: native settings, no SAVIOR-Train fine-tune.

6. Failure modes

  • Ours (SAVIOR-Train fine-tuned): multilingual documents; ambiguous

very-dense layouts.

  • Competitors / zero-shot panel: vertical text; fine print;

multi-column layouts; logo-embedded text; degraded scans; handwriting; structured headers; bold-styled text.

  • Residual: the multilingual + very-dense-layout cluster persists even

after fine-tuning.
